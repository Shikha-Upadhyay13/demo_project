# AdaptivAI Mini — Agentic Course Generator

Type any tech topic. Get a full mini-course generated by an agent.

A demo of agentic coding: LangChain + LangGraph + LangSmith doing what each does best.

## What it does

1. Catalog page (Google Skills layout) shows pre-generated + on-demand courses.
2. Type a topic → an agent finds YouTube videos, fetches transcripts, embeds them, writes a summary, builds a 5-question quiz — live, in under a minute.
3. Course page (NotebookLM layout) shows videos + transcripts with a side panel for Summary, Chat (RAG grounded in transcripts, with citations), and Quiz.

## Run

```bash
python -m venv .venv
.venv\Scripts\activate              # Windows
# source .venv/bin/activate         # macOS/Linux

pip install -r requirements.txt
cp .env.example .env                # then fill in real keys

# Sanity check the Course Builder end-to-end (no server yet):
python scripts/smoke_test.py "Docker basics"

# Start the app:
uvicorn app:app --reload --port 8000
# open http://localhost:8000
```

## Night-before: pre-generate seed courses

```bash
python scripts/pregen_courses.py
```

## Architecture (two graphs)

- **Course Builder** (one-shot, heavy): `video_finder → transcript_fetcher → chunker_embedder → {summary_generator ∥ quiz_generator} → END`
- **Tutor Chatbot** (per-message, light): `router → {retrieve_and_answer | direct_answer | clarify} → END`

They share a Chroma collection keyed by `course_id`. They do not share state.
